# 插圖資源指南

## 下載插圖

請前往 [unDraw.co](https://undraw.co/illustrations) 下載以下插圖，並設定主色為 **#E08E6D**（暖橘色）。

### 需要的插圖清單

1. **roommates.svg** - 用於 Carousel 第一張（主 CTA）
   - 搜尋關鍵字: "roommates" 或 "moving"
   - 建議尺寸: 300-400px 寬
   - 用途: 傳達找室友的概念

2. **people_search.svg** - 用於「我們如何幫你找到合適室友？」區塊
   - 搜尋關鍵字: "people search" 或 "community"
   - 建議尺寸: 200-250px 寬
   - 用途: 傳達配對機制的概念

3. **friends.svg** - 用於配對案例參考區塊
   - 搜尋關鍵字: "friends" 或 "happy announcement"
   - 建議尺寸: 250-300px 寬
   - 用途: 傳達成功配對的溫馨感

4. **empty.svg** - 用於「還沒有您的配對資料」彈窗
   - 搜尋關鍵字: "empty" 或 "searching"
   - 建議尺寸: 150-200px 寬
   - 用途: 友善地提示用戶尚未註冊

## 下載步驟

1. 訪問 https://undraw.co/illustrations
2. 在搜尋框輸入上述關鍵字
3. 點擊右上角調色盤圖示，輸入 **#E08E6D**
4. 下載 SVG 格式
5. 將檔案重新命名為上述檔案名稱
6. 放入此目錄 (`public/illustrations/`)

## 替代資源

如果 unDraw 找不到合適的插圖，也可以使用：
- [Humaaans](https://www.humaaans.com/) - 可自定義的人物插圖
- [Open Doodles](https://www.opendoodles.com/) - 手繪風格插圖
- [DrawKit](https://www.drawkit.io/) - 免費 SVG 插圖

## 顏色設定

所有插圖的主色都應該設定為：
- **主色**: #E08E6D (暖橘色)
- **輔助色**: 保持預設或使用 #6EE0A8 (淺綠色)
